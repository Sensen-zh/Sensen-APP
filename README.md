#项目介绍
森森软件库，这是一个基于PHP+JSON开发的软件库程序
前端+后端使用PHP+HTML5 数据使用JSON 更简单方便
不使用任何框架模板等 纯原TML5自主编写
后端支持修改系统设置，发布软件，修改软件，模板切换，在线更新
使用蓝白风格搭配制作 简约好看的搭配
